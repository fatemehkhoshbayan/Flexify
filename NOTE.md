# Flexify — smart workout search and challenges

## Project pitch

**Flexify** is a web app that helps people discover exercises and build personalized workout plans. It targets anyone who wants a simple, interactive way to browse workouts by muscle group and difficulty — without a heavy fitness platform or expensive subscriptions.

---

## How this project came together

These are the main steps I took while building Flexify:

1. **Idea and scope**  
   Started by searching for a clear concept: a lightweight tool focused on search, filters, and planning — not a full gym management system.

2. **Data and backend**  
   Looked for exercise APIs and data-fetching options; many good sources were paid. I ended up creating JSON that matches the shape I need, implementing a **Nest** backend, deploying it on **Vercel**, and fetching that data from this frontend.

3. **Visual design**  
   Researched a **motivating, energetic color palette** so the UI feels upbeat and fitness-oriented without feeling cluttered.

4. **Imagery**  
   **Generated muscle images** (and related visuals) so exercise cards and details feel consistent and readable at a glance.

5. **Frontend architecture**  
   Built the experience as a **single-page application (SPA)** so navigation between exercises, favorites, and plan flows stays fast and smooth.

---

## One-line summary

Flexify: discover exercises, save favorites, and request a plan — powered by your own API and data, with a bold look and a SPA shell.
